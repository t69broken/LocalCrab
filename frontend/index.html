📖 Setup Guide

Step 1 — Create a bot
  1. Open Telegram and search for @BotFather
  2. Send the command /newbot
  3. Follow the prompts — choose a name and username (must end in bot)
  4. BotFather will reply with your bot token
Step 2 — Find your Telegram user ID
  1. Message @userinfobot on Telegram
  2. It will instantly reply with your numeric user ID
  3. Paste it into the Allowed User IDs field above
  4. Add more people by separating IDs with commas
Step 3 — Connect & chat
  1. Paste the token and your user ID above, then click Save & Connect
  2. Open Telegram and start chatting with your bot
  3. The bot talks to your default agent by default
Bot commands
/start — greeting and help
/agents — list all LocalCrab agents
/agent <id> — switch to a specific agent
/clear — reset current agent context
/status — GPU and model info
Any other message — chat with the active agent
Persistence note: Settings are saved to /app/data/comms_config.json and survive container restarts. You can also set TELEGRAM_BOT_TOKEN and TELEGRAM_ALLOWED_USERS in your .env file as a fallback.
